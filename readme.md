# node-jquery-extend

A tiny toolkit, based on [node-jquery-lite](https://github.com/phonowell/node-jquery-lite).

## Install

```
$ npm i node-jquery-extend
```

## Usage

```
$ = require 'node-jquery-extend'
$.i 'Hello world!'
```

## Methods

- `$.get()`
- `$.i()`
- `$.info()`
- `$.log()`
- `$.next()`
- `$.parseJson()`
- `$.parsePts()`
- `$.parseSafe()`
- `$.parseShortDate()`
- `$.parseString()`
- `$.parseTemp()`
- `$.post()`
- `$.shell()`
- `$.serialize()`
- `$.timeStamp()`
