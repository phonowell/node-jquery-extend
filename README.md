# node-jquery-extend
A tiny toolkit, based on [node-jquery-lite](https://github.com/phonowell/node-jquery-lite).

## Install
```
$ npm i node-jquery-extend
```

## Methods
- `$.i()`
- `$.info()`
- `$.log()`
- `$.next()`
- `$.parseJson()`
- `$.parsePts()`
- `$.parseSafe()`
- `$.parseShortDate()`
- `$.parseString()`
- `$.parseTemp()`
- `$.shell()`
- `$.timeStamp()`

This project is not **STABLE**.
